Detailed Description

template<typename T>
class CMesh< T >

This class holds both the geometrical data of the mesh and the discrete data on it. It is also responsible for setting the initial conditions and printing the data. Currently, the only output implemented is a basic .txt format. Thus, it is not advisable to output large amounts of data.

void CMesh< T >::set_initial_conditions ( F  f)
inline

The function is used to initialize the field variable values with the initial conditions required by means of calling function "f" that can be passed as an argument. For example, if your function is called "my_inital_conditions", you need to invoke this with set_initial_conditions(my_inital_conditions). Your function should not modify the field size, but only fill in the values.
