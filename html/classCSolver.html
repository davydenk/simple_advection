This function applies the first order upwind scheme to the mesh data. The basic scheme info can be found in en.wikipedia.org/wiki/Upwind_scheme. Parameters used:

  • a reference to a mesh variable
  • advection equation coefficient
  • time step
  • the number of steps
  • PritInterval for the output, if set to 0, no intermidiate output occurs.

Currently, the output is performed in text format, so it is not adviced to print too many steps. The intermidiate output files are written to the "gif_dir" folder with the names "TIME_STEP.dat". The final output is printed to the "result.txt" file.
